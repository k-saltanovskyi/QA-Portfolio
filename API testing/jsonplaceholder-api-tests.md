# API Testing - JSONPlaceholder

**URL:** https://jsonplaceholder.typicode.com/users

**Ожидаемый результат:** Статус код 200, возвращается список пользователей в JSON формате

**Фактический реультат:** Статус код 200, список пользователей получен 

**Статус:** Pass
