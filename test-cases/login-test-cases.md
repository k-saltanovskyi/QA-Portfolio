# Тест-кейсы для функции входа в систему

## TC-001 — Вход с правильными данными
**Предусловие:** Пользователь зарегистрирован, приложение открыто

**Шаги:**
1. Ввести правильный логин
2. Ввести правильный пароль
3. Нажать "Войти"

**Ожидаемый результат:** Пользователь попадает в личный кабинет

**Фактический результат:** Пользователь попал в личный кабинет 

**Статус:** Pass 

---

## TC-002 — Вход с неправильным паролем
**Предусловие:** Пользователь зарегистрирован, приложение открыто

**Шаги:**
1. Ввести правильный логин
2. Ввести неправильный пароль
3. Нажать "Войти"

**Ожидаемый результат:** Система показывает ошибку "Неверный логин или пароль"

**Фактический результат:** Система показала ошибку "Неверный логин или пароль" 

**Статус:** Pass
